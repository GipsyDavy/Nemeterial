# Gestor de Materiales (Nemeterial / App de préstamo de equipamiento)

Aplicación para gestión de préstamo y devolución de material corporativo (vehículos, móviles, cámaras, drones, llaves, etc.).

## Tecnologías

- **Frontend móvil** → Android (Kotlin/Java) – carpeta `app-android`
- **Backend**       → Spring Boot (Java) – carpeta `spring-boot`
- **Base de datos** → MySQL + SQLite (según entorno) – carpeta `database`
- **Documentación** → diagramas de flujo, requisitos – carpeta `docs`
